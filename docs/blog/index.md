# Research blog

Every week, an automated pipeline sweeps PubMed, Europe PMC, Semantic Scholar,
and bioRxiv for new work relevant to *SAMHD1* p.A565T and its mechanism, scores
each paper against this exact phenotype, and publishes the digest here. On the
first of each month, a synthesis post summarizes what moved.

Posts are tagged by mechanistic pathway (cGAS-STING, NLRP3, VDAC1,
ISG15-mitophagy, JAK-STAT, gene editing, and others) so you can follow a single
thread through the literature. Subscribe via the **RSS icon** in the top bar.

!!! info "How to read the scores"
    Each paper carries a 0–10 relevance score. **7–10** is high relevance
    (directly on-mechanism); **5–6** is adjacent; **3–4** is a possible
    specialty bridge worth a glance. Scores are generated by a language model
    against a fixed phenotype profile and are a triage aid, not a judgment of
    scientific quality.
